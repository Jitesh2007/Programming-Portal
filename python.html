Python Tutorial

Hello! Welcome to Python. Python is a beginner-friendly programming language. We will learn the basics, how to work with variables, loops, functions, and write small programs.

1. Printing Text

print("Hello, World!")

This will show "Hello, World!" on the screen.

2. Variables

# Numbers
age = 18

# Text
name = "Jitesh"

# Boolean
is_student = True

3. Data Types

Python has common data types:

number = 10       # int
decimal = 3.14     # float
text = "Hello"     # str
flag = True        # bool

4. Lists

Lists store multiple values:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple

5. Loops

Loops help repeat actions:

for i in range(5):
    print("Number:", i)

6. Functions

Functions are reusable code blocks:

def greet(name):
    print("Hello", name)

greet("Jitesh")

7. Conditional Statements

age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

8. Simple Program

Let’s combine concepts:

name = input("Enter your name: ")
print("Welcome,", name)

for i in range(3):
    print("This is loop number", i + 1)

Practice these examples and try changing the values. Python becomes fun when you run the code and see results immediately!
