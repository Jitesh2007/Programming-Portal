Java Tutorial

Welcome to Java. Java is a popular, secure, and object-oriented programming language. It is used to build Android apps, web applications, and enterprise software.

1. First Java Program

class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World");
    }
}

2. Why Java?

Java is platform independent, meaning it can run on any system. Java follows the concept of "Write Once, Run Anywhere".

3. Variables and Data Types

int age = 18;
double marks = 85.5;
char grade = 'A';
String name = "Jitesh";

4. Input Using Scanner

import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int num = sc.nextInt();
System.out.println(num);

5. Conditional Statements

int age = 20;

if (age >= 18) {
    System.out.println("Eligible to vote");
} else {
    System.out.println("Not eligible");
}

6. Loops

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

7. Methods (Functions)

static void greet() {
    System.out.println("Welcome to Java");
}

public static void main(String[] args) {
    greet();
}

8. Classes and Objects

class Student {
    String name;
    int age;
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "Jitesh";
        s1.age = 18;

        System.out.println(s1.name + " " + s1.age);
    }
}

9. Simple Addition Program

int a = 5;
int b = 10;
System.out.println("Sum = " + (a + b));

Practice Java regularly. Java builds strong programming fundamentals and is very useful for real-world applications.
