CSS Tutorial

Hi! In this page, I’ll guide you to learn CSS step by step. CSS is what makes your web page look nice and colorful. You’ll learn how to style text, arrange content, and make your site look professional.

1. How to Add CSS

We can style HTML in three ways:

<!-- Inline CSS -->
<p style="color:red;">Hello!</p>

<!-- Internal CSS -->
<style>
  p { color: red; }
</style>

<!-- External CSS -->
<link rel="stylesheet" href="style.css">

2. Selecting Elements

You can choose which elements to style using selectors:

/* By tag name */
p { color: blue; }

/* By class */
.box { padding: 10px; }

/* By ID */
#main { background: #f0f0f0; }

3. Colors & Fonts

Let’s make things colorful and easy to read:

body {
  background-color: #f9f9f9;
  color: #333;
  font-family: 'Poppins', sans-serif;
}

h1 {
  color: #0072ff;
}

4. The Box Model

Every element is like a box. It has content, padding, border, and margin:

div {
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

5. Layout with Flexbox

Flexbox helps arrange elements easily:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

6. Make It Responsive

We can adjust layouts for mobile screens:

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

7. Styling Links

a {
  color: #0072ff;
  text-decoration: none;
}

a:hover {
  color: #005bb5;
}

Keep practicing! CSS is all about trying things out and seeing how it looks. You’ll get better each day.
