HTML Tutorial

Hello! I’m going to guide you step by step to learn HTML. HTML is what gives structure to your web page. You’ll learn to add headings, paragraphs, images, links, lists, and more. Let’s start!

1. Basic Structure

Every HTML page starts like this:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

2. Headings

HTML has 6 heading levels: <h1> to <h6>. Use h1 for main titles, h2 for subtitles, etc.

<h1>Main Title</h1>
<h2>Subtitle</h2>

3. Paragraphs

Paragraphs are simple text blocks wrapped in <p>:

<p>This is a paragraph.</p>

4. Links

You can link to other pages or websites with <a>:

<a href="https://example.com">Visit Example</a>

5. Images

Images make your page more attractive:

<img src="myimage.png" alt="Description">

6. Lists

You can make ordered and unordered lists:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

7. Forms

Forms let users input data:

<form>
  <label>Name:</label>
  <input type="text">
  <input type="submit" value="Submit">
</form>

Keep experimenting! HTML is easy if you practice. Add headings, paragraphs, images, and see how it looks in your browser.
