C Language Tutorial

Hello! Welcome to the C programming language. C is a powerful and fast language. It is used to build operating systems, drivers, and many core programs. Let’s learn C step by step.

1. First C Program

This is a simple C program:

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

2. Structure of C Program

#include <stdio.h> → Used for input/output functions
main() → Program execution starts here
printf() → Prints output on screen

3. Variables

Variables store data:

int age = 18;
float marks = 85.5;
char grade = 'A';

4. Data Types

int number = 10;
float price = 99.99;
char letter = 'C';

5. Input from User

int age;
scanf("%d", &age);
printf("Age is %d", age);

6. Conditional Statements

int age = 18;

if (age >= 18) {
    printf("You can vote");
} else {
    printf("You cannot vote");
}

7. Loops

For loop example:

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

8. Functions

void greet() {
    printf("Welcome to C");
}

int main() {
    greet();
    return 0;
}

9. Simple Program

int a, b;
printf("Enter two numbers:");
scanf("%d %d", &a, &b);
printf("Sum = %d", a + b);

Practice writing C programs regularly. C builds strong programming logic and helps you understand how computers work internally.
