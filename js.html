JavaScript Tutorial

Hey! Welcome to JavaScript. This is the language that makes your web pages interactive. You’ll learn variables, functions, loops, and how to manipulate your webpage dynamically.

1. Adding JavaScript

You can add JS in three ways:

<!-- Inline -->
<button onclick="alert('Hi!')">Click Me</button>

<!-- Internal -->
<script>
  alert('Hello World!');
</script>

<!-- External -->
<script src="script.js"></script>

2. Variables

Variables store data:

// Numbers
let age = 18;

// Text
let name = "Jitesh";

// Boolean
let isStudent = true;

3. Functions

Functions are reusable blocks of code:

function greet() {
  alert("Hello!");
}

greet(); // call the function

4. Loops

Loops help repeat tasks:

for (let i = 1; i <= 5; i++) {
  console.log("Number: " + i);
}

5. DOM Manipulation

Change webpage content dynamically:

document.getElementById("demo").innerText = "Hello JS!";

Example in HTML:

<p id="demo">Original Text</p>
<button onclick="document.getElementById('demo').innerText = 'Changed!'>Change Text</button>

6. Event Listeners

document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});

Practice these examples and try changing values. JS becomes fun when you see changes live on your page!
