C++ Tutorial

Hello! Welcome to C++. C++ is an extension of C with Object-Oriented Programming (OOP) features. It is used in software development, games, and competitive programming.

1. First C++ Program

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!";
    return 0;
}

2. Difference Between C and C++

C++ supports classes and objects, while C does not. C++ allows better code organization using OOP concepts.

3. Variables

int age = 18;
float marks = 90.5;
char grade = 'A';
string name = "Jitesh";

4. Input and Output

int num;
cin >> num;
cout << "Number is " << num;

5. Conditional Statements

int age = 18;

if (age >= 18) {
    cout << "Eligible to vote";
} else {
    cout << "Not eligible";
}

6. Loops

for (int i = 1; i <= 5; i++) {
    cout << i << endl;
}

7. Functions

void greet() {
    cout << "Welcome to C++";
}

int main() {
    greet();
    return 0;
}

8. Classes and Objects

class Student {
public:
    string name;
    int age;
};

int main() {
    Student s1;
    s1.name = "Jitesh";
    s1.age = 18;

    cout << s1.name << " " << s1.age;
    return 0;
}

9. Simple Program

int a, b;
cout << "Enter two numbers: ";
cin >> a >> b;
cout << "Sum = " << a + b;

Practice writing programs regularly. C++ is very powerful and helps you understand advanced programming concepts.
